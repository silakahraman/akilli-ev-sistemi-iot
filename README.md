# 🏠 Akıllı Ev Sistemi (ESP32 Tabanlı IoT Projesi)

Bu proje, ESP32 mikrodenetleyicisi kullanılarak geliştirilen IoT tabanlı bir akıllı ev sistemidir. Kullanıcılar web arayüzü veya mobil cihaz üzerinden ev içerisindeki ışık, perde ve kapı gibi bileşenleri uzaktan kontrol edebilmektedir.

> **Not:** Bu proje takım çalışması kapsamında geliştirilmiştir.

---

# 📌 Proje Amacı

Bu proje kapsamında ESP32 tabanlı, internet üzerinden kontrol edilebilen küçük ölçekli bir akıllı ev sistemi geliştirilmiştir.

SolidWorks ile ev modeli tasarlanmış, Arduino IDE kullanılarak gömülü sistem yazılımı geliştirilmiş ve kullanıcı dostu bir kontrol arayüzü oluşturulmuştur.

---

# 🚀 Proje Özellikleri

- Web arayüzü üzerinden kontrol
